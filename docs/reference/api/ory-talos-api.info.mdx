Ory Talos is a high-performance API key management service. It handles the full API key lifecycle:
issuing keys, verifying them at low latency, deriving short-lived tokens (JWT and Macaroon), and
revoking access.

`Admin*`-prefixed RPCs require admin authentication. The bare `RevokeAPIKey` RPC is authenticated by
proof of possession (the raw credential secret).

### Quick Links

- [**Admin**](./admin-issue-api-key) — key lifecycle management (issue, rotate, revoke, import,
  derive tokens) and verification (single and batch)
- [**Public**](./get-jwks) — JWKS endpoint for public key distribution (RFC 7517)
- [**Self-service**](./revoke-api-key) — proof-of-possession self-revocation for credential holders
